About This Project

This e-commerce app is a work in progress. The goal is to build a modern, responsive Angular storefront with essential features like browsing products, adding to cart, and a smooth checkout flow.

Current Status

  • Home page with product listing: Done
  • Product cards with add/remove to cart: Done
  • Persistent cart and cart page: Done
  • Checkout form and order summary: In Progress
  • Payment gateway integration: Planned
  • Styling and responsiveness: Ongoing

Roadmap

Upcoming tasks include real payment integration (Stripe/Razorpay/PayPal), order confirmation emails, user accounts, and an admin dashboard for product management.

If you'd like help implementing any feature or want to contribute, feel free to open an issue or reach out.
